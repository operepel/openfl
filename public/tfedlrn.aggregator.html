tfedlrn.aggregator.aggregator module¶

class tfedlrn.aggregator.aggregator.Aggregator(id, fed_id, col_ids, connection, init_model_fpath, latest_model_fpath, best_model_fpath)¶

Bases: object

An Aggregator is the centra node in federated learning.

Parameters

  • id (str) – Aggregation ID.

  • fed_id (str) – Federation ID.

  • col_ids (list of str) – The list of IDs of enrolled collaborators.

  • connection – Used to be ZMQ connection, but deprecated in gRPC.

  • init_model_fpath (str) – The location of the initial weight file.

  • latest_model_fpath (str) – The file location to store the latest weight.

  • best_model_fpath (str) – The file location to store the weight of the best model.
